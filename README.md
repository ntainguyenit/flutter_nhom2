# Mobile App Flutter

## Giới thiệu

**Mobile App Flutter** là ứng dụng tổng hợp các **Flutter** được xây dựng trong quá trình học môn **Lập trình Mobile**.  

Ứng dụng sử dụng **Flutter** để xây dựng giao diện và quản lý điều hướng giữa các màn hình thông qua **Drawer Menu & Navigation**.

Mỗi bài tập được thiết kế như **một màn hình (Screen) độc lập**, cho phép người dùng truy cập nhanh thông qua **Menu**.

---

## Công nghệ sử dụng

- **Flutter**
- **Dart**
- **Material Design**
- **Navigator & Routes**
- **Drawer Menu**
- **REST API (News API)**

## Điều hướng ứng dụng

- Ứng dụng sử dụng **Drawer Menu** để hiển thị danh sách
- Mỗi màn hình đều có:
  - **Menu** để chuyển nhanh giữa các danh sách
  - **Nút về Trang chủ**
- Điều hướng giữa các màn hình bằng **Navigator & Routes**

## Hướng dẫn chạy project

### 1️ Clone repository

```bash
git clone https://github.com/ntainguyenit/Mobile_App_Flutter.git
```

### 2️ Cài đặt các package

```bash
flutter pub get
```

### 3️ Chạy ứng dụng

```bash
flutter run
```
